Справочная информация к «Система моделирования движения транспорта на автодороге»

Настройка количества полос движения

Ниже представлена форма для выбора типа движения и количества полос движения:

Рисунок 1 — Форма настройки параметров дороги


На форме следует выбрать количество направлений движения и с помощью ползунков задать количество полос в каждом направлении.

Возможность выбора количества полос для встречного движения появится только в случае, если количество направлений движения равно двум.

При нажатии на кнопку 'Назад' текущее окно закроется и откроется форма настройки типа дороги. При нажатии на кнопку 'Далее' будет открыто окно для выбора распределения интенсивности потока автомобилей.
