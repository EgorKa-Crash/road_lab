Ниже поедставлены изображения формы, по настройке интенсивности автомобильного потока.

Рисунок 2 — Форма настройки интенсивности потока с равномерным законом распределения


Рисунок 3 — Форма настройки интенсивности потока с нормальным законом распределения


Рисунок 4 — Форма настройки интенсивности потока с показательным законом распределения


На форме следует выбрать желаемый тип распределения для потока машин.

После выбора типа распределения станут доступны ползунки задания параметров случайной/постоянной величины.

При нажатии кнопки 'Назад' текущее окно закроется и откроется форма выбора количества полос или настройка светофорной фазы. При нажатии кнопки 'Далее' будет открыто окно настройки скоростного режима.
