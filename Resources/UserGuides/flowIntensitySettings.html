Настройка интенсивности потока

Ниже представлены изображения формы для настройки интенсивности транспортного потока.

Рисунок 1 — Форма для настройки параметра детерминированного транспортного потока


Рисунок 2 — Форма для настройки параметров равномерного транспортного потока


Рисунок 3 — Форма для настройки параметров нормального транспортного потока


Рисунок 4 — Форма для настройки параметра экспоненциального транспортного потока


На форме следует выбрать желаемый тип распределения транспортного потока.

После выбора типа распределения станут доступны ползунки для задания параметров случайной/постоянной величины.

При нажатии на кнопку 'Назад' текущее окно закроется и откроется форма для выбора количества полос или настройка светофорной фазы. При нажатии на кнопку 'Далее' будет открыто окно для настройки скоростного режима.
