Настройка скоростного режима

Ниже представлены изображения формы для настройки скоростного режима автомобилей.

Рисунок 1 — Форма для настройки параметра детерминированного скоростного режима


Рисунок 2 — Форма для настройки параметров равномерного скоростного режима


Рисунок 3 — Форма для настройки параметров нормального скоростного режима


Рисунок 4 — Форма для настройки параметра экспоненциального скоростного режима


На форме следует выбрать желаемый тип распределения скоростного режима автомобилей.

После выбора типа распределения станут доступны ползунки для задания параметров случайной/постоянной величины.

При нажатии на кнопку 'Назад' текущее окно закроется и откроется форма для настройки интенсивности транспортного потока. При нажатии на кнопку 'Далее' будет открыта главная форма программы.
